# Panel Logos

Upload your panel logos here with the following naming convention:

- `panel-1.png` - First panel logo
- `panel-2.png` - Second panel logo
- `panel-3.png` - Third panel logo
- `panel-4.png` - Fourth panel logo
- `panel-5.png` - Fifth panel logo
- `panel-6.png` - Sixth panel logo
- `panel-7.png` - Seventh panel logo
- `panel-8.png` - Eighth panel logo
- `panel-9.png` - Ninth panel logo
- `panel-10.png` - Tenth panel logo

## Image Requirements

- **Format**: PNG, JPG, or WebP
- **Recommended size**: 400x400 pixels (square)
- **Background**: Transparent PNG recommended
- **File size**: Keep under 200KB for optimal loading

## After Uploading Logos

Once you've uploaded the logos:

1. Update the panel names in `/components/PanelKlinik.tsx`
2. The logos will automatically display on the website
3. No code changes needed - just replace the image files!

## Current Setup

The component is set to display placeholders until you upload the actual panel logos.
